# javascript-fetch-api-demo

This is a simple JavaScript project that demonstrates how to use the Fetch API to retrieve data from an external API and display it dynamically on a web page.

## 🚀 Features
- Fetch data from Fake Store API
- Convert response into JSON format
- Dynamically render product cards
- Display product image, title, price, rating, and category

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript (Fetch API)

## 📌 What I Learned
- How to use `fetch()` to get API data
- Handling Promises using `.then()`
- Converting response using `.json()`
- Dynamically updating DOM using JavaScript

## 🔗 API Used
https://fakestoreapi.com/products

## ▶️ How to Run
1. Download or clone the repository
2. Open `index.html` in your browser
